Java_Board_Project

프로젝트 설명
----------------------------------------------------------------------------------------------------------------------------------------------
* 자바와 데이터베이스 사이를 자동 매핑 기능을 지원하는 mybatis를 이용해서 mariaDB에 저장하고 게시판 CRUD를 만들어보는 프로젝트입니다.
